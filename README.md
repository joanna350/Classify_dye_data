# XGBoost

## To Run

#### Command Line Interface

```
# For description
python_v __scriptname -h

# For operation
python_v __scriptname__ arg1 arg2 arg3'
```
* `plot_cnf`: arg1 (default 0)
* `gridsearch`: arg2 (default 0)
* `others`: arg3 (default 1)
